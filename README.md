# Smart Waste Management System

A full-stack waste Management System

-**Frontend: **React.js
-**Backend: **Flask(Python)
-**Database: **MySQL
-**Charts: **Chart.js


## Features
-Admin & Worker roles
-Live bin status tracking
-Full bin alerts
-Collection tracking(prototype)
-Interactive dashboard & charts
